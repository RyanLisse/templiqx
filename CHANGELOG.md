# Changelog

All notable changes to Templiqx are documented here. The project follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and uses Semantic
Versioning for tagged releases.

## [Unreleased]

### Added

- Production-release and full mock-conformance hardening in progress.

## [0.2.0] - Unreleased

### Changed

- Coordinated the engine and all SDK packages from `0.1.0` to `0.2.0` with engine API line `0.2`.
- The Operations API changed additively with new operation IDs `assessQualityProposals`; no operation IDs were removed.

## [0.1.0] - Unreleased

Initial standalone, provider-neutral AI interaction contract compiler POC with
Rust, CLI, and MCP surfaces; deterministic CRM3 conformance; bounded DOCX V5 and
HTML rendering adapters; Docker/Compose/Helm smoke paths; and supply-chain gates.
